# PLATT2-Override

The software for the VEXU Team Plattifornia (PLATT2). This codebase is being used for the 2026-2027 VEX Robotics season Override.

## Getting Started

### Dependencies

* PROS 4.2.2
* Eigen 3.4.90

### Installing

This project can be ran using either the PROS VSCode extension or with the PROS CLI. 

### PROS VSCode Extension
* Open the PLATT2-Override folder in VSCode with the PROS extension enabled and do the following
   * Open the integrated terminal
   * Run the ```pros m``` command
   * To depoloy to a VEX V5 brain, run ```pros mu```
